expectation/mean

for discrete X

\displaystyle E[g(X)] = \sum_k g(k)P(X=k)

for continuous X with density function f(x)

\displaystyle E[g(X)] = \int_{-\infty}^{\infty} g(x)f(x)dx

'nth moment' is E[X^n]

variance

variance formula in terms of expectation

\displaystyle \text{Var}(X) = E[X^2] - E[X]^2

expectation is like linear ofzo

\displaystyle E[aX + b] = aE[X] + b

variance some identity

\displaystyle \text{Var}(aX + b) = a^2\text{Var}(X)
